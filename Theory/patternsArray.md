For **array problems specifically**, most questions fall into a smaller set of patterns. These are the **main array patterns you should know**:

1. **Prefix Sum**
2. **Two Pointers**
3. **Sliding Window**
4. **Running Minimum / Maximum (Greedy scan)**
5. **Kadane’s Algorithm (Maximum Subarray)**
6. **Binary Search on Array**
7. **Sorting + Scan**
8. **Hash Map / Frequency Counting**
9. **Monotonic Stack**
10. **Monotonic Queue (Deque)**

These **10 patterns cover the majority of array problems** you'll see on platforms like LeetCode or HackerRank.

If you learn to recognize these patterns, most array problems become **much easier to categorize and solve**.
